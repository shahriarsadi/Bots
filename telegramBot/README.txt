This is a bot that I built on telegram platform. 
To run this bot:
 you must have node.js installed 
 you must have npm.js installed 
 Then install the node-telegram-bot-api 
 you also need a bot token for the bot you can do this by visiting telegram and messaging the Bot Father and follow the steps
To use the bot simply: 
 type /start 
 and follow the instructions
